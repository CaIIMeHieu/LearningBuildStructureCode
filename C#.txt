Lifetime

	Transition : Tạo mới một instance với mỗi instance trong DI container 
	Scope : Tạo mới một instance cho mỗi scope ( thường là mỗi request trong web app ), trong DI container có nhiều instance đi nữa thì cũng chỉ có một instance được tạo ra trong mỗi scope
	Singleton : Tạo mới một instance duy nhất cho toàn bộ ứng dụng, tất cả các request sẽ dùng chung instance này

Convention mặc định của Exeception handler là Transition
